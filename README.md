# Apache Knox (apache-knox)
Apache Knox is a REST API and application gateway for the Apache Hadoop ecosystem. It provides a single access point for all REST and HTTP interactions with Apache Hadoop clusters, with authentication, authorization, and audit capabilities.

## APIs

### Apache Knox
Knox provides a gateway REST API that proxies and secures access to Hadoop ecosystem services (HDFS, Hive, HBase, Oozie, etc.), with topology definition APIs, token management APIs, and service discovery for dynamic cluster integration.
